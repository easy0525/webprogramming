소개

최소한의 코드로 빠르게 배포할 수 있도록 설계했습니다. 다크모드 자동 지원, 접근성 포커스 링, OG 메타, 반응형 레이아웃 포함.

🚀 배포 리포지토리 → Settings → Pages → Deploy from a branch → main 선택
🖼 OG 이미지 og-image.png 파일을 루트에 두고 메타 태그 URL을 본인 주소로 교체
